# SEO Implementation - Phase 3+: Further Enhancements

## 概要 (Overview)
「他にもできること」というご要望に応え、サイト全体の「サブページ（下層ページ）」のSEOを強化しました。
Topページだけでなく、料金ページやプロフィールページなどが検索結果に表示された際も、クリック率が高まるように最適化しています。

## 実施内容 (Changes)

### 1. Subpage Metadata Optimization
各ページごとに固有の `description` と `title` を設定し、検索エンジンがページ内容を正しく理解できるようにしました。

- **料金・エリア (`src/pages/price.astro`)**
    - `Title`: 料金・エリア | enak - **たまプラーザ・青葉区・宮前区**
    - `Description`: 「たまプラーザ・横浜市青葉区・川崎市宮前区のベビーシッター料金...」
- **サービス内容 (`src/pages/service.astro`)**
    - `Title`: サービス内容 | enak - **知育遊び・産後ケア**
    - `Description`: 「手形アートや小麦粉粘土などの知育遊び...」
- **プロフィール (`src/pages/profile.astro`)**
    - `Title`: プロフィール | enak - **保育士 小林睦**
    - `Description`: 「たまプラーザ・横浜市青葉区のベビーシッターenak代表、小林睦のプロフィール...」

### 2. Image Alt Text Optimization
検索エンジンの画像検索（Google画像検索など）からの流入を意識し、主要な画像の `alt` 属性を具体化しました。

- **プロフィール写真 (`src/pages/profile.astro`)**
    - `Before`: "Profile Photo"
    - `After`: "**たまプラーザのベビーシッター 小林睦（横浜市青葉区・宮前区）**"

## 推奨アクション (Recommended Actions - Non-Code)

技術面以外で、**最も効果が高いもの**は以下のGoogleマップ対策です。

### 📍 Googleビジネスプロフィール (MEO対策)
「たまプラーザ ベビーシッター」で検索した際、地図枠（ローカルパック）に表示されるためには、Googleビジネスプロフィールへの登録が必須です。
1.  **Googleビジネスプロフィールに登録**（無料）
2.  **エリア設定**: 「非店舗型（出張型）」として登録し、対象エリアを「横浜市青葉区」「川崎市宮前区」に設定。
3.  **オーナー確認**: 郵送ハガキ等でオーナー確認を完了させる。
4.  **写真投稿**: サービス中の様子やアイコンなどを投稿。

これはコード修正では対応できないため、ぜひ実施をおすすめします。
